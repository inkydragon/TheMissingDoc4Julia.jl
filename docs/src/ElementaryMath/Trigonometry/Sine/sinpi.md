# `sinpi`

Compute the sine of `Ï€*x`, where `x` expressed in half-revolutions.

```@setup repl_only
using LinearAlgebra
```
```@docs
Base.sinpi
```


## Methods

```@repl
methods(sinpi, (Any,), [Base, Base.Math, Base.MathConstants, Base.MPFR])
```


## Examples

### Real Numbers
```jldoctest
julia> sinpi(0)
0.0

julia> sinpi(0.5)
1.0

julia> sinpi(1)
0.0
```

### Complex
```jldoctest
julia> sinpi(0+0im)
0.0 + 0.0im
```
