# `tanpi`

Compute the tangent of `Ï€*x`, where `x` expressed in half-revolutions.

```@setup repl_only
using LinearAlgebra
```



## Methods

```@repl
methods(tanpi, (Any,), [Base, Base.Math, Base.MathConstants, Base.MPFR])
```


## Examples

### Real Numbers
```jldoctest
julia> tanpi(0)
0.0

julia> tanpi(-0.0)
-0.0

julia> tanpi(0.5)
Inf

julia> tanpi(1.0)
-0.0

julia> tanpi(2.0)
0.0
```

### Complex
```jldoctest
julia> tanpi(0+0im)
0.0 + 0.0im
```
