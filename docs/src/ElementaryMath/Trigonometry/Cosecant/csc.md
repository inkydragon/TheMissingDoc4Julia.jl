# `csc`

Compute the cosecant of `x` expressed in radians.

```@setup repl_only
using LinearAlgebra
```
```@docs
Base.csc
```


## Methods

```@repl
methods(csc, (Any,), [Base, Base.Math, Base.MathConstants, Base.MPFR])
```


## Examples

### Real Numbers
```jldoctest
julia> csc(0)
Inf

julia> csc(-0.0)
-Inf

julia> csc(pi/2)
1.0

julia> csc(pi)
Inf
```

### Complex
```jldoctest
julia> csc(0+0im)
NaN + NaN*im
```
