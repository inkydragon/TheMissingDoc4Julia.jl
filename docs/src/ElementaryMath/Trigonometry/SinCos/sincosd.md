# `sincosd`

Compute the sine and cosine of `x` expressed in degrees.

```@setup repl_only
using LinearAlgebra
```
```@docs
Base.sincosd
```


## Methods

```@repl
methods(sincosd, (Any,), [Base, Base.Math, Base.MathConstants, Base.MPFR])
```


## Examples

### Real Numbers
```jldoctest
julia> sincosd(0)
(0.0, 1.0)

julia> sincosd(-0.0)
(-0.0, 1.0)

julia> sincosd(45)
(0.7071067811865476, 0.7071067811865476)

julia> sincosd(90)
(1.0, 0.0)

julia> sincosd(180)
(0.0, -1.0)
```

### Complex
```jldoctest
julia> sincosd(0+0im)
(0.0 + 0.0im, 1.0 - 0.0im)
```
