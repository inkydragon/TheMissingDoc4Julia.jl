# `acoth`

Compute the arc hyperbolic cotangent of `x`.

```@setup repl_only
using LinearAlgebra
```
```@docs
Base.acoth
```


## Methods

```@repl
methods(acoth, (Any,), [Base, Base.Math, Base.MathConstants, Base.MPFR])
```


## Examples

### Real Numbers
```jldoctest
julia> acoth(Inf)
0.0

julia> acoth(-Inf)
-0.0

julia> acoth(1.313)
1.0000487358456431
```

### Complex
```jldoctest
julia> acoth(Inf+0im)
0.0 - 0.0im
```

## Tech Notes

- Implemented in terms of [`atanh`](@ref): `acoth(x) = inv(atanh(x))`
