asecd

Compute the principal value of the arc secant of x, return in degrees.

Base.Math.asecd — Function
asecd(x)

Compute the inverse secant of x, where the output is in degrees. If x is a matrix, x needs to be a square matrix.

Julia 1.7

Matrix arguments require Julia 1.7 or later.

source

julia> methods(asecd, (Any,), [Base, Base.Math, Base.MathConstants, Base.MPFR])# 1 method for generic function "asecd" from Base.Math:

Examples

Real Numbers

julia> asecd(1.0)
0.0

julia> asecd(sqrt(2))
45.00000000000001

julia> asecd(2.0)
60.00000000000001

julia> asecd(Inf)
90.0

julia> asecd(-1.0)
180.0

Complex

julia> asecd(1+0im)
0.0 + 0.0im
