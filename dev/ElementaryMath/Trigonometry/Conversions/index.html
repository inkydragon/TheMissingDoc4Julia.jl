Conversions

Base.Math.rad2deg — Function
rad2deg(x)

Convert x from radians to degrees.

See also deg2rad.

Examples

julia> rad2deg(pi)
180.0
source
Base.Math.deg2rad — Function
deg2rad(x)

Convert x from degrees to radians.

See also rad2deg, sind, pi.

Examples

julia> deg2rad(90)
1.5707963267948966
